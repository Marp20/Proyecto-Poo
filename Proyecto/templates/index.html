Proyecto POO

Las funcionalidades presentes son:

#Login: permite iniciar sesion de un usuario de tipo Preceptor o Padre.

#Registrar Asistencia: al usuario Preceptor le permite cargar en la base de datos la Asistencia de los alumnos de un curso determinado

#Informe con detalles: al usuario Preceptor le permite visualisar los datos cargados por la funcion Registrar Asistencia.
